# react-app
使用webpack自行搭建react环境，使用react 16.4 + react-router 4.3 + typescript + mobx 实现antd-pro
\
使用node + mock 模拟数据

## 启动项目
```sh
安装依赖
$ npm install
启动项目
$ npm run start
在浏览器中输入 http://localhost:8888
```

## 项目目录
```sh
components // 自定义组件
datasources // 自定义数据类
pages // 项目的页面view
routes // 项目的路由定义
store // redux的定义
 --actions //redux的action定义
 --reducers // redux的reducer定义
utils // 项目自定义工具集合
```
